# Avalon Hotels (Next.js)

Premium, light/white, luxury-style hotel website starter built with **Next.js App Router** + TypeScript.

## Quick start

1) Install dependencies
```bash
npm install
```

2) Run locally
```bash
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
- Push this repo to GitHub
- Import it into Vercel and deploy
- Add your custom domain in Vercel → Settings → Domains

## Cloudbeds booking link
Edit the booking engine URL in:
`src/lib/config.ts`
